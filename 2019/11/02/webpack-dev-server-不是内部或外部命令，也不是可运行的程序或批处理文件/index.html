webpack-dev-server 不是内部或外部命令，也不是可运行的程序或批处理文件

发表于 2019-11-02 分类于 webpack 阅读次数： Valine：
本文字数： 1.1k 阅读时长 ≈ 1 分钟 
【运行报错:】webpack-dev-server 不是内部或外部命令，也不是可运行的程序或批处理文件

Yarn 的全称是 Yet Another Resource Negotiator，意思是“另一种资源调度器”，是一个新的快速安全可信赖的可以替代NPM的依赖管理工具，由 facebook 推出并开源。

运行项目显示缺少必要的插件

运行npm下载

npm install webpack-dev-server --save

安装成功之后再次运行

yarn install

结果还是报错

索性删除项目目录下的node_modules文件夹

npm 的工作是把将各种各样的包拉到您的计算机上，并且进行管理，这是 Nodejs 周边生态的重要体现，而node_modules文档就是存放这些安装包的地方。

然后重新运行

yarn install

再运行

yarn install

成功运行。
